🧠 AI Chatbot (Flask + Gemini + Voice)

A simple AI-powered chatbot web app built with Flask and Google Gemini AI, featuring chat memory, voice output, and offline fallback support.

🚀 Features

💬 AI chat using Google Gemini API

🧠 Chat memory with SQLite

🎙️ Voice replies using gTTS

🔄 Offline fallback responses

📜 Session-based chat history

🌐 Simple Flask web UI

🛠️ Tech Stack

Python

Flask

Google Gemini API

SQLite

gTTS

HTML
